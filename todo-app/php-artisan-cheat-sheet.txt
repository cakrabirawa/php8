# 📄 CHEATSHEET LARAVEL ARTISAN

## 📌 1. Perintah Dasar & Utilitas
* php artisan list          - Menampilkan semua perintah artisan yang tersedia.
* php artisan help [cmd]    - Melihat detail cara menggunakan suatu perintah.
* php artisan about         - Melihat ringkasan informasi aplikasi, versi Laravel, dan PHP.
* php artisan tinker        - Masuk ke REPL interaktif untuk uji coba kode PHP/Eloquent.

## 🗄️ 2. Database & Migrasi (Migration)
* php artisan make:migration [nama] - Membuat file migrasi baru.
* php artisan migrate               - Menjalankan semua migrasi yang belum dieksekusi.
* php artisan migrate:rollback      - Membatalkan (undo) migrasi terakhir.
* php artisan migrate:reset         - Membatalkan semua migrasi dari awal.
* php artisan migrate:refresh       - Membatalkan semua migrasi, lalu menjalankannya ulang.
* php artisan migrate:fresh         - Menghapus paksa semua tabel, lalu migrasi dari awal.
* php artisan db:seed               - Menjalankan pengisian data (seeding) ke database.
* php artisan db:seed --class=[Nama]- Menjalankan seeder spesifik.

## 🏗️ 3. Pembuatan Komponen (Make Commands)
* php artisan make:model [Nama]     - Membuat file Eloquent Model.
* php artisan make:model [Nama] -mcr- Membuat Model sekaligus Migration, Controller, & Resource.
* php artisan make:controller [Nama]- Membuat Controller kosong.
* php artisan make:controller -r    - Membuat Controller dengan fungsi CRUD lengkap.
* php artisan make:seeder [Nama]    - Membuat file Database Seeder.
* php artisan make:factory [Nama]   - Membuat Model Factory untuk data palsu (testing).
* php artisan make:middleware [Nama]- Membuat HTTP Middleware baru.
* php artisan make:request [Nama]   - Membuat file validasi Form Request.

## 🛣️ 4. Rute (Routes)
* php artisan route:list            - Menampilkan daftar semua rute yang terdaftar.
* php artisan route:list --path=[kw]- Mencari rute spesifik berdasarkan URL.
* php artisan route:cache           - Membuat cache rute untuk mempercepat performa produksi.
* php artisan route:clear           - Menghapus cache rute.

## ⚡ 5. Performa & Cache
* php artisan config:cache          - Menggabungkan semua konfigurasi menjadi satu file cache.
* php artisan config:clear          - Menghapus cache konfigurasi (wajib setelah ubah .env).
* php artisan cache:clear           - Mengosongkan cache aplikasi.
* php artisan view:cache            - Mengompilasi semua file Blade view.
* php artisan view:clear            - Menghapus kompilasi file Blade view.
* php artisan optimize              - Menjalankan cache konfigurasi dan rute sekaligus.
* php artisan optimize:clear        - Menghapus semua file cache optimasi.

## 🌐 6. Server Lokal
* php artisan serve                 - Menjalankan server lokal (default: http://127.0.0.1:8000).
* php artisan serve --port=[nomor]  - Menjalankan server pada port spesifik (misal: 8080).
